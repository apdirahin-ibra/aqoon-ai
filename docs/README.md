# Aqoon AI Documentation

Welcome to the internal engineering documentation for Aqoon AI. This directory is organized to separate high-level concepts from step-by-step guides and API references.

## Folder Structure

*   **/architecture**: High-level system design documents, Architectural Decision Records (ADRs), and deep dives into how systems connect (e.g., frontend to Convex).
*   **/api**: (Upcoming) Detailed API references and schema definitions for the backend logic.
*   **/guides**: (Upcoming) Step-by-step tutorials and playbooks for developers (e.g., local setup, deployment guides, testing procedures).
*   **/assets**: Image files, diagrams, and flowcharts specifically used within these markdown files.

> **Tip:** If you're looking for an overview of the project or instructions on how to run it locally, please see the [Root README](../README.md).
